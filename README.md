# 儿童生长发育评估小程序

基于**首都儿科研究所**《中国0~18岁儿童、青少年身高、体重的标准化生长曲线》（中华儿科杂志，2009年）数据开发的微信小程序。

## 功能特点

### 📊 身高评估
- 输入年龄、性别、身高
- 计算精确百分位（P3-P97之间精确到个位）
- 使用**线性插值法**计算更精确的百分位

### ⚖️ 体重评估
- 输入体重后自动计算百分位
- 结合身高百分位综合评估BMI

### 🩺 血压评估（可选）
- 输入收缩压和舒张压
- 基于《中国儿童青少年血压参考标准》评估
- 给出正常/偏高/高血压判断

## 数据来源

| 数据项 | 来源 |
|--------|------|
| 身高/体重百分位 | 首都儿科研究所《0-18岁儿童青少年身高、体重百分位数值表》 |
| BMI评估 | 同上，结合身高体重综合判断 |
| 血压正常值 | 《中国儿童青少年血压参考标准》 |

## 百分位计算说明

本小程序使用**线性插值法**计算精确百分位：

例如：6岁女孩体重为19kg
- P25 = 18.0kg，P50 = 20.0kg
- 19kg落在P25和P50之间
- 计算：(19-18)/(20-18) = 0.5
- 精确百分位：25 + (50-25)×0.5 = **P37**
